# python.dev-bootstrap

```bash
curl -fsSL 'https://raw.githubusercontent.com/abx-ltd/python.dev-bootstrap/main/lib/dev-bootstrap/bootstrap.py' | python3
```

Pin a release by changing the right-hand side to `env RELEASE_TAG='releases/gh/0.3.9' python3` (same `curl` URL). If this file is hosted elsewhere, replace the `raw.githubusercontent.com` URL accordingly.
